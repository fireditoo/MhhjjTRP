# 项目简介
前端框架采用 Vue 3.5 + Vite 6 构建，使用组合式 API 开发。UI 组件库选用 Element Plus 2.9，配合 @element-plus/icons-vue 图标库。状态管理使用 Pinia 3，路由管理使用 Vue Router 4。数据可视化采用 ECharts 5 渲染图表。样式方面使用 SCSS 预处理器。后端采用springboot，数据库mysql8

有三个端
社长端拥有最高管理权限，可进行成员管理（添加、编辑、删除成员，审批加入/退出申请，设置身份标签）、活动全流程管理（创建活动、审批活动、管理报名签到）、经费管理（登记收支、审批报销、查看台账）、物资管理（入库、领用、归还、报废）、公告管理（发布、编辑、删除通知）以及查看数据报表。

社员端功能相对精简，主要包括查看首页仪表盘和待办事项、浏览公告通知并标记已读、报名参加活动、申请领用和归还物资、查看个人参与记录。

指导老师端侧重监督和审批，可查看社团运营数据报表、审批重要活动和经费申请、查看成员信息和活动记录、监督社团整体运营状况。
系统通过角色选择实现权限隔离，不同角色登录后加载对应的菜单和功能模块。

基于springboot+vue3的校园社团管理系统是一个旨在提高校园社团管理效率的系统。该系统采用前后端分离的架构，前端框架采用 Vue 3.5 + Vite 6 构建，后端采用 springboot 和 mysql8 数据库。

## 核心功能
- 成员管理：包括添加、编辑、删除成员，审批加入/退出申请，设置身份标签。
- 活动管理：创建活动、审批活动、管理报名签到。
- 经费管理：登记收支、审批报销、查看台账。
- 物资管理：入库、领用、归还、报废。
- 公告管理：发布、编辑、删除通知。
- 数据报表：查看社团运营数据报表。

## 适用场景
- 校园社团内部管理
- 提高社团运营效率
- 便于成员参与和交流

## 技术特点
- 前端：Vue 3.5 + Vite 6，Element Plus 2.9 UI 组件库，Pinia 3 状态管理，Vue Router 4 路由管理，ECharts 5 数据可视化，SCSS 预处理器。
- 后端：springboot，mysql8 数据库。
- 角色权限隔离：通过角色选择实现权限隔离，不同角色登录后加载对应的菜单和功能模块。
